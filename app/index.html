A game jam is an event where programmers, artists and gamers come together and create games; both computer games and board games. ​
At our game jam there is a 48 hours time limit and a randomized theme for inspiration.
Visit our Facebook page for upcoming events!

Who are welcome?

Everybody is welcome, student and non-students alike. No previous experience is needed and the event is free of charge.

Where is it?

The event will be held at CreActive in Linköping. There will be wireless internet, free tea, hot chocolate and coffee.

FAQ

  • Is there somewhere I can sleep?
    It is not possible to sleep at the event, sorry.
  • Is there something I can eat?
    There will be free snacks at the site. Bring cash, there will be breakfast buffet at a low price. We will also order pizza on friday night and asian food in the early afternoon on saturday.
  • Do I need to be part of a team before the jam?
    There usually are plenty of people looking for a group at the Game Jam site and we will help you find a group if you cannot find one yourself.

Great tools

Tools for making computer games
  • Unity 3D (C# / Javascript)
  • Construct 2 (Visual scripting)
  • pyGame (Python)
  • Love 2D (Lua)
  • Unreal Engine (C++ / Visual Scripting)
  • Haxeflixel (Haxe)
  • Scratch (Visual Scripting)
Tools for drawing
  • Blender 3D
  • MagicaVoxel (Voxel editor)
  • Krita
Tools for making sound effects
  • bfxr (Online old school sound effect generator)
  • Sunvox (Modular synthesizer / tracker for music)
  • BeepBox (Online chiptune music tracker)
Free game resources
  • OpenGameArt.org (Public domain and creative commons game assets)
Tools for keeping track of work
  • Git
    • GitLab
    • GitHub
    • IDA GitLab
  • SourceTree (frontend for Git)
  • Git kraken (cross platform frontend for Git)
